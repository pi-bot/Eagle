# Eagle
Eagle Resources for the PiBot

## Component List Conventions

- U = chip or integrated circuit
- S = Switch
- R = Resistor
- C = Capacitor
- D = Diode
- F = Fuse
- H = Header connector
- J = connector
- L = inductor or LED
- X = crystal

### Other conventions not used:
- B = battery
- K = relay
- Q = transistor
- T = transformer
- V = varistor / MOV
-BR = Bridge Rectifier
- M = MOSFET
- J = jack (connector)
- P = plug
